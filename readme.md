vtm is a text-based desktop environment.

# Supported platforms

- Windows
  - Windows 8.1 or later
- Unix
  - Linux
  - macOS
  - FreeBSD
  - NetBSD
  - OpenBSD
  - [`...`](https://en.wikipedia.org/wiki/POSIX#POSIX-oriented_operating_systems)
